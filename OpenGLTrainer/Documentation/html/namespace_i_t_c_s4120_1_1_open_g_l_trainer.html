'matchWorldCoordinatesToPixelCoordinates' is a GLUT reshape callback that updates OpenGL view transformations so that OpenGL world coordinates directly map to pixel coordinates in the Window's framebuffer with (0,0) at the lower-left and (width-1,height-1) at the upper- right where width and height are the current size of Window sub-window.

Usage:

int main (int argc, char** argv)
    {
    glutInit();
        ...
    glutReshapeFunc(OpenGLTrainer::matchWorldCoordinatesToPixelCoordinates);
        ...
    glutMainLoop();
    }
Note:
ITCS4120: For early assignments, students can just use this function without worrying about how it works.
void ITCS4120::OpenGLTrainer::showFrameRate ( )

When placed at end of your display callback (but before any call to glFlush to glutSwapBuffer), 'showFrameRate' displays the framerate in the lower left corner of the GLUT window. 

void display(void)
    {
    ...

    ITCS4120::OpenGLTrainer::showFrameRate();
    glSwapBuffer(); // if using double buffering 
    }
Warning:
Uses standard clock() timer whose limited resolution may not yield very accurate results
Doesn't support multiple GLUT Windows (uses static local variables)
Not Thread Safe - Uses static local variables
