Compilation Instructions

MSVS 2010

By default the OpenGLTrainer source distribution assumes OpenGLTrainer exists in the a directory structure as follows:

  • "Third Party Libraries\OpenGLTrainer\" - root directory of OpenGLTrainer source distribution founds at https://cci-subv.uncc.edu/svn/zwartell-public/ITCS 6120-8120/OpenGLTrainer/trunk
  • "Third Party Libraries\glew" - GLEW forked distribution found at https://cci-subv.uncc.edu/svn/zwartell-public/vendor/glew/local/current 
  • "Third Party Libraries\glut" - Glut forked distribution found at https://cci-subv.uncc.edu/svn/zwartell-public/vendor/opengl.org/glut/local/current
  • "Third Party Libraries\FreeImage" - FreeImage forked distribution found at https://cci-subv.uncc.edu/svn/zwartell-public/vendor/FreeImage/local/current 

For each and any of the above libraries that you do not already have, create a svn externals link on your "Third Party Libraries" to retrieve the indicate repo redirectory into the indicated subdirectory in your "Third Party Libraries" directory.  Also, the environment variable UNCC_ITCS_GRAPHICS must be set to the directory directly above the "Third Party Libraries" mentioned above.

To compile, first compile OpenGLTrainer's dependencies as follows:

  1.   Open "Third Party Libraries\glew\build\vc10\glew.sln" and compile the Project glew_shared using Configuration Debug
  2.   Open "Third Party Libraries\glut\make\Windows\MSVS 10\glut.sln" and compile the Project glut using Configuration MT_Debug_DLL
  3.   Open "Third Party Libraries\FreeImage\FreeImage.2010.sln" and compile Project FreeImagePlus using Configuration Debug

Next compile OpenGLTrainer as follows:

  • Open "OpenGLTrainer\make\MSVS 10\OpenGLTrainer.sln" and compile the Configuration Debug

As a final test compile and run the PanZoomWindow application:
  1. Open ""OpenGLTrainer\Tools and Examples\PanZoomWindow\make\MSVS 10\PanZoomWindow.sln"
  2. Compile and run (see instructions in PanZoomWindow/README.txt)

Others

Support for all other compilers is deprecated at this time although various old compilation scripts are possibly still hanging around.

Compilation Scripts

Audience: Maintainers of OpenGLTrainer

MSVS Property Sheets (.props files) are located in "OpenGLTrainer\make\MSVS 10\props".   There are two basic .props hierarchies: one for OpenGLTrainer Maintainers and one for OpenGLTrainer Applications developers.  The two hierarchy's share some (but not all) .props files.  By default for applications this is done using the svn externals links.  Application's retrieve the default set of .props files from the OpenGLTrainer SVN repository at $OPENGLTRAINER_SVN_REPO/trunk/make/MSVS 10/props/.


    • OpenGLTrainer Applications:     The .props files are as follows. 

      • OpenGLTrainer - Debug.props - An OpenGLTrainer application's vcxproj's Debug config Properties must include this file. 
        • File Location:   MyApp\make\MSVS 10\props\OpenGLTrainer\        [via svn external link]

        • Inheritance:   This .props file automatically inherit from the following:

          • OpenGLTrainer Dependencies - Debug.props - contains Debug config specific settings.
            • File Location:   MyApp\make\MSVS 10\props\OpenGLTrainer\             [via svn external link]

          • OpenGLTrainer.props - contains generic config settings (common to all config's). 
            • User Macro's for file paths to OpenGLTrainer and it's dependent library's files
            • other common compiler settings

            • File Location:   MyApp\make\MSVS 10\props\OpenGLTrainer\             [via svn external link]

          • OPENGLTRAINER_DIR.props - contains 1 User Macro OPENGLTRAINER_DIR which is path to root of OpenGLTrainer source distribution
            • Application developers must set this one User Macro.   All other User Macro's in the other .props file build off of this one. Each app contains

            • File Location:   MyApp\make\MSVS 10\                             [local copy; not a svn external link]

    • OpenGLTrainer Maintainer:     The MSVS Property file hierarchy is as follows:

      • OpenGLTrainer Lib - Debug.props - Debug config specific settings used for compiling OpenGLTrainer. This file inherits the follow prop's files:
        • OpenGLTrainer Dependencies - Debug.props - see above
          • File Location:   OpenGLTrainer\make\MSVS 10\props             [local copy; svn linked to by apps]

        • OpenGLTrainer.props - see above
          • File Location:   OpenGLTrainer\make\MSVS 10\props             [local copy; svn linked to by apps]

        • OPENGLTRAINER_DIR.props - contains 1 User Macro OPENGLTRAINER_DIR which is path to root of OpenGLTrainer source distribution
          • File Location:   OpenGLTrainer\make\MSVS 10\                      [local copy, not svn link'ed by app's]
