OpenGL and GLUT - 101

Location:  OpenGLTrainer\Tools and Examples\OpenGL and GLUT - 101

After compiling OpenGLTrainer, open:

    "OpenGL and GLUT - 101\make\MSVS 10\OpenGL and GLUT - 101.sln"

and compile and run.

This application opens a small OpenGL window.  Pressing and holding the left mouse button drags a small red rectangle around the screen.  Various debugging output is displayed on the console window.   See the source code for further details.
