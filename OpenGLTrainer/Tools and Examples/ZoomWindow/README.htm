The ZoomWindow program captures the screen image and redisplays in a separate window that can be panned and zoomed.    

  • Left-click dragging the mouse pans the contents of the zoom window. 
  • Right-click dragging up/down zooms in and out using the original mouse position as the center  of room.
  • Middle-click brings up pop-up menu. 
    • The 'Capture' item enables/disables active capturing of the target window/screen.

Limitations:

  • ZoomWindow isn’t particularly efficient.   It uses the Win32 GDI API to copy the screen framebuffer into host memory and then copies from host memory into GL texture memory.   Presumably some use of GL Pixel Buffer Object’s, etc could keep all the data on the graphics card.
  • In dual monitor systems ZoomWindow only grabs the image from the primary monitor.
